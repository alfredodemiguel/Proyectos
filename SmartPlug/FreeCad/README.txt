# Summary

This box can be customized in FreeCad, just by changing some parameters.

1) open the attached FreeCad file
2) in FreeCad, open the spreadsheet "Parameters"
3) adjust the values (height, width...) as required

4) if you need, do not hesitate to add your changes (ie add support for power bank, add a hole for an USB connector...)

5) before converting to Mesh, set the value of Parameters / Lid / Rotation to 180 (to have the lid on the correct position on to the printer bed). If you don't do that, you will have to rotate the lid in your slicer

6) go to workbench Mesh Design
7) select the body
8) Menu Meshes / Create Mesh from shape 
9) Menu Meshes / Export Mesh (=> here you have your body.stl file)

10) select the Lid
11) Menu Meshes / Create Mesh from shape 
12) Menu Meshes / Export Mesh (=> here you have your Lid.stl file)

Hope this box will be useful for some of you.
